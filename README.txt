Minimal package to test various features of package creation in R

Defining S4 classes, S4 and S3 methods, documentation via roxygen, testing via testthat.

Compiling the package:
- copy the entire set of folders
- use createPackage_packMin.R in the coding folder
- this will create packMin (the package), compile and check it

Version 1 uploaded on 2017/01/03
